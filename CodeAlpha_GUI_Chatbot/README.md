# AI Chatbot - Personalized Swing Chatbot

## Overview

AI Chatbot is a simple Java-based chatbot developed using Java Swing. The application provides an interactive graphical user interface (GUI) where users can communicate with the chatbot through text messages.

The chatbot uses a rule-based Natural Language Processing (NLP) approach by matching user input with predefined keywords and generating appropriate responses.

This project was developed as part of the CodeAlpha Java Programming Internship.

---

## Features

* Interactive GUI built using Java Swing
* Rule-based chatbot responses
* Basic NLP using keyword matching
* Personalized greeting for the user
* User-friendly chat interface
* Object-Oriented Programming (OOP) implementation
* Lightweight and easy to run

## File Description

### Main.java

Entry point of the application. Launches the chatbot GUI.

### Chatbot.java

Contains chatbot logic and predefined responses using HashMap.

### ChatbotGUI.java

Creates the graphical user interface and handles user interactions.
